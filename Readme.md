# EA Tech Radar

## Installation

Run `npm i`

## Develop

Run `npm start` then open a browser to [localhost:4000](http://localhost:4000)
 
## Deploy

Nix-based: `NODE_ENV=production npm run build`

Win-based: `set NODE_ENV=production npm run build`

Assets will be compiled to `/dist`
