# EA Tech Radar

## Installation

Run `npm i`

## Develop

Run `npm start` then open a browser to [localhost:4000](http://localhost:4000)
 
## Build

`npm run build`

Assets will be compiled to `/dist`

## Deploy



## Features

- Critical for inlining CSS
- Modular D3
- React with Redux
- Fetch & Promises
- Publish to Surge & Github Pages
